# Functions providing reports to be used in imongr

Functions providing reports to be used in imongr

## Usage

``` r
.registry_status_data(pool)

registry_status_report(pool, pool_verify)
```

## Arguments

- pool:

  Database pool object for connection to the production system

- pool_verify:

  Database pool object for connection to staged data

## Value

A data frame holding the report data
