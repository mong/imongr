# User example data for qmongr data store

Meta data to allow interactions with qmong data. Only used for testing
in this package. Please refere to the overall data model (ref)

## Usage

``` r
user
```

## Format

Data frame with made up values

- user_name:

  String user name of the person altering data, *e.g.* 'jesusc'

- user:

  String full name of the person altering data, *e.g.* 'Jesus Christ'

- phone:

  String phone number of the person altering data, *e.g.* '+4747474747

- email:

  String e-mail address of the person altering data, *e.g.*
  'jesusc\\sky.com'

- valid:

  Numeric definig if user is active (1) or not (0)
