# Example data for qmongr data store

Meta data to allow interactions with qmong data. Only used for testing
in this package. Please refere to the overall data model (ref)

## Usage

``` r
data
```

## Format

Data frame with made up values

- Aar:

  Numeric 4 digit year

- ShNavn:

  String hospital name

- ReshId:

  String internal id of hospital

- OrgNrShus:

  Numeric 9 digit id of hospital

- Variabel:

  String variable name

- kvalIndID:

  Numeric id of quality indicator

- delivery_id:

  Numeric id referencing the delivery
